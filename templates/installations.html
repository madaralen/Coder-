How to Install Coder Bot

  1. Visit the GitHub App installation page
  2. Click "Install" button
  3. Select the repositories you want to enable the bot for
  4. Grant the required permissions
  5. Start mentioning @coder-bot in issues or PRs

Required Permissions

Coder Bot requires the following permissions to function properly:

Repository Permissions

  • Issues (Read & Write)
  • Pull Requests (Read & Write)
  • Contents (Read & Write)
  • Metadata (Read)

Account Permissions

  • Email addresses (Read)

Events

  • Issues
  • Pull Requests
  • Issue Comments
  • Pull Request Reviews
